# homebrew-mommy
homebrew formula for [mommy](https://github.com/FWDekker/mommy)~ ❤️

see [mommy](https://github.com/FWDekker/mommy) for installation instructions~

## development
to update the formula to mommy version `<version>`, run `./update.sh <version>`~
